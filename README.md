<p align="center">Work in progress!</b></p>

Used technologies:

  - Angular
    - Material UI
    - CKEditor

Used applications:

- WebStorm
- Google Chrome
- Adobe Photoshop

Check also ZeroOne-TicketingSystem-Backend repository for more informations about backend layer created in Java 17 with Spring Boot framework.
